7-GUIs implementation in Vanilla JavaScript

This project was and is not an attempt to pioneer new GUI programming paradigms, or to really comply with existing ones. To be honest I don't really know what I'm doing haha, think of this project as me throwing things at something until it works (with a bit of learning on the side).

When I first came across Eugen Kiss' 7-GUIs: A GUI programming benchmark, The implications of the challenge weren't transparent to me. I came across the challanges, and I just wanted to try build them. It looked like fun.

Coming into the project I only had pre-existing knowledge of how to build the first two challanges (so no pre-existing knowledge really), as I write this im currently on the 6th challenge— Circle Drawer, and I can confidently say amongst my mess I've picked up a few things!
